# Kamote-Racer-2D

## Project Description
Kamote Racer 2D is a nostalgic, retro-styled 2D racing game that revives classic arcade fun for the modern era.  
Dodge traffic, rack up points, and see how long you can survive on the endless highway.

## Features
- Classic top-down pixel-art visuals  
- Smooth keyboard & optional drag-and-drop controls  
- Progressive difficulty—traffic gets faster the longer you survive  
- High-score tracker so you can beat the best run  

## How to Play
1. Launch the game.  
2. Use the **Arrow Keys** (← ↑ ↓ →) to steer your car.  
   - *Optional:* drag-and-drop the car if you prefer touch/mouse control.  
3. Avoid colliding with oncoming traffic.  
4. A single collision ends your run—stay sharp!
5. Survive as long as possible to climb the leaderboard!

## Controls
| Action        | Key / Input        |
|---------------|--------------------|
| Move Left     | ← Left Arrow       |
| Move Right    | → Right Arrow      |
| Accelerate    | ↑ Up Arrow         |
| Brake/Reverse | ↓ Down Arrow     |
| Drag Car      | Click & Drag       |

**Good luck, and happy racing!**
